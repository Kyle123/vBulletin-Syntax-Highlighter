Syntax Highlighter

A syntax highlighter for VBA on MrExcel, ExcelForum, ozGrid and Excel Fox
(also works on fomulas)

Forum Highlighting

This adds syntax highlighting to your code, you can customize the colours and everyone can see the results.

You can change the colours below and preview the results. If you’ve already got a forum window open when you change and save colours, you’ll need to refresh it before your changes are applied.

Preview

Personal Highlighting

This adds syntax highlighting to all code, whether it’s posted by you or another user.

Only you see this highlighting, other users see what is displayed above.

You can turn this on or off.

Preview
